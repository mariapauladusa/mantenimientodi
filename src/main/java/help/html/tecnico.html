Pantalla Técnico

Pulsando F1 en la pantalla principal, se abrira esta pagina de ayuda.

En esta pantalla el usuario que entre como tacnico podra, ver todas las incidencias de todos los profesores, añadir nuevas incidencias, modificarlas y enviar correos a otros miembros del profesorado

Acciones es un Menu Bar donde se situa esta ayuda, la pantalla de enviar correo y por ultimo añadir una nueva incidencia

Haciendo click derecho sobre una fila de la tabla apareceran las opciones de modificar una incidencia o borrarla

En el boton de consultar tipos apareceran los tipos de estado o urgencia existentes para que el usuario pueda buscar incidencias segun el estado o la urgencia que esta introduzca en el espacio para el texto

El botón de Actualizar Información es para volver a mostrar todas las incidencias despues de haber buscado alguna/as en concreto segun estado o urgencía
