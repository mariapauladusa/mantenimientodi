Pantalla Administrador

Pulsando F1 en la ventana principal, se abrira esta pagina de ayuda.

El usuario que entre como administrador, podra: ver todas las incidencias de todos los profesores, insertar nuevas incidencias, modificarlas, enviar correos, ver estadistica de las incidencias consultar incidencias segun estado o urgencia y si no se acuerda del tipo de un estado o urgencia lo podra consultar.

Haciendo click derecho sobre una incidencia seleccionada podra modificar o borrar la incidencia seleccionada

Otra accian que podra realizar el usuario que entre como administrador sera la de ver todos los miembros del profesorado, modificar su informacion, dar de baja a un miembro del profesorado y exportar o importar datos de la tabla o a la tabla
